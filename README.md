# M98.NET  
  M98の.NET版です。  
  
[概要]  
  M98.NETは乱数をフル活用した自動作曲ソフトウェアです。  
  これは古代さんのご厚意でM98を.NET向けに移植したものです。  
  
[機能、特徴]  
 ・オリジナルのM98はAMC98(mucom88の後継FM音源ドライバ)向けでしたが、  
 本ソフトウェアはmucom88.NETで動作するよう調整しています。  
 M98.NETはmucom88の.mucファイルを基準に動作し、専用コマンドで指定された動作に沿ってMMLデータを書き換えます。  
 ・coreプログラムをdllに分けたコンソールアプリです。  
  
[必要な環境]  
 ・Windows7以降のOSがインストールされたPC  
 ・テキストエディター  
 ・気合と根性
